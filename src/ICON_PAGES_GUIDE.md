# 우정 타로 - 아이콘별 세부 페이지 가이드

## 📱 네비게이션 아이콘 소개

상단 네비게이션 바의 4개 아이콘이 모두 실제 페이지와 연결되었습니다!

---

## 1. 👤 프로필 (Profile)

### 접근 방법
- 홈 화면 상단 네비게이션의 프로필 버튼 클릭
- 프로필이 있는 경우: 프로필 뷰 페이지로 이동
- 프로필이 없는 경우: 프로필 입력 모달 표시

### 주요 기능

#### 프로필 헤더
- **아바타**: 성별에 맞는 이모지 표시 (👦/👧/⭐)
- **이름 & 별자리**: 생년월일 기반 자동 계산
- **퀵 스탯**: 나이, 생일, 친구 이름 한눈에 확인

#### 성격 분석 상태
- ✅ **완료**: 사주 & 성격 분석 페이지 접근 가능
- ⚠️ **미완료**: 성격 테스트 진행 안내

#### 계정 정보
- 가입일
- 타로 리딩 횟수
- 데이터 저장 방식 (로컬 저장소)

#### 성취 배지
- 우정 탐험가 배지 표시
- 사용자의 노력 인정 메시지

#### 액션
- **프로필 수정하기** 버튼으로 정보 업데이트

---

## 2. 🕐 히스토리 (History)

### 접근 방법
- 홈 화면 상단 네비게이션의 히스토리 버튼 클릭

### 주요 기능

#### 통계 대시보드
- 📊 **전체 리딩**: 총 타로 리딩 횟수
- 📈 **이번 달**: 이번 달 리딩 횟수
- 📋 **인기 주제**: 가장 많이 본 질문 카테고리

#### 히스토리 리스트
- **월별 그룹화**: 날짜별로 깔끔하게 정리
- **질문 & 요약**: 어떤 고민에 대한 리딩이었는지
- **카드 프리뷰**: 3장 리딩 (과거-현재-가능성) 미리보기
  - ⏪ 과거 카드
  - ⏺️ 현재 카드
  - ⏩ 가능성 카드

#### 인터랙션
- 카드 클릭 시 상세 내용 보기 (준비 중 알림)
- 호버 효과로 직관적인 UX

#### 데이터
- 현재는 샘플 데이터로 구성
- 실제 사용 시 자동 저장 기능 추가 예정

## 3. 📖 가이드 (Guide)

### 접근 방법
- 홈 화면 상단 네비게이션의 가이드 버튼 클릭

### 주요 기능

#### 3가지 탭 구성

##### 1️⃣ 사용 가이드 탭
**타로란 무엇인가요?**
- 타로의 기본 개념 설명
- Friendship Tarot의 특징 안내
- 메이저 아르카나 22장 사용
- 긍정적 해석 철학

**타로 리딩 방법**
- 4단계 프로세스:
  1. 질문 선택
  2. 카드 선택
  3. 해석 확인
  4. 행동하기

**3장 리딩의 의미**
- ⏪ **과거**: 관계의 시작과 영향
- ⏺️ **현재**: 현재 상태와 상황
- ⏩ **가능성**: 미래의 방향성

**타로를 대하는 마음가짐**
- 가능성 vs 절대적 미래
- 열린 마음으로 수용
- 주체적 선택
- 성장의 기회

##### 2️⃣ 전체 카드 탭
- 22장 메이저 아르카나 카드 전체 보기
- 검색 기능으로 원하는 카드 찾기
- 각 카드별:
  - 카드 이름 (한글 & 영문)
  - ☀️ 정방향 해석
  - 🌙 역방향 해석

##### 3️⃣ 메이저 아르카나 탭
- 메이저 아르카나만 필터링
- 동일한 카드 그리드 표시

#### 카드 상세 모달
- 카드 클릭 시 전체 정보 모달
- 정방향/역방향 해석 상세 보기
- 아름다운 애니메이션 효과

## 4. ⚙️ 설정 (Settings)

### 접근 방법
- 홈 화면 상단 네비게이션의 설정 버튼 클릭

### 주요 기능

#### 외관 섹션
**다크 모드**
- 현재 다크 테마 활성화
- 타로 감성을 위한 최적화된 디자인

**컬러 팔레트**
- 4가지 주요 색상 표시
  - 💜 Purple (#6B46C1)
  - 🟡 Gold (#F59E0B)
  - 💗 Pink (#EC4899)
  - 💜 Violet (#8B5CF6)

#### 알림 섹션
**푸시 알림**
- 일일 타로 리마인더
- 토글 스위치로 ON/OFF

**사운드**
- 카드 효과음 및 배경음악
- 토글 스위치로 ON/OFF

#### 일반 섹션
**언어**
- 현재: 한국어
- 다국어 지원 준비 중

**자동 저장**
- 타로 리딩 결과 자동 저장
- 토글 스위치로 ON/OFF

#### 정보 & 지원
- 📚 **도움말**: 가이드 보기 (준비 중)
- 🛡️ **개인정보 처리방침**: 보기 (준비 중)
- ℹ️ **앱 버전**: 1.0.0
- 🌐 **개발자**: Figma Make

#### 데이터 관리
- ⚠️ **모든 데이터 삭제**
  - 프로필 & 히스토리 완전 삭제
  - 확인 팝업으로 실수 방지
  - 되돌릴 수 없는 작업

## 🎨 디자인 특징

### 일관된 레이아웃
- 모든 페이지에 "돌아가기" 버튼
- 헤더에 이모지 + 제목 + 설명
- 통일된 Glass Card 디자인

### 애니메이션
- Framer Motion 활용
- 부드러운 페이드인 효과
- 리스트 항목별 순차 애니메이션
- 호버 효과로 인터랙티브

### 반응형
- 데스크톱/태블릿/모바일 최적화
- 자동 그리드 레이아웃
- 유연한 카드 배치

### 색상 시스템
- 섹션별 구분 색상
  - 프로필: 골드
  - 히스토리: 퍼플
  - 가이드: 핑크
  - 설정: 그린/레드

## 🔄 사용자 플로우

### 첫 방문 사용자
1. 홈 화면 진입
2. 프로필 버튼 → 프로필 입력
3. 가이드 버튼 → 타로 사용법 학습
4. 질문 선택 → 타로 리딩
5. 히스토리 버튼 → 과거 리딩 확인

### 재방문 사용자
1. 홈 화면 진입 (환영 메시지)
2. "나의 사주 & 성격 분석 보기" 클릭
3. 새로운 타로 리딩
4. 히스토리에서 과거 리딩 복습
5. 프로필 보기로 성장 확인
