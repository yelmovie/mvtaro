# 마음코칭 카드 - 프로필 & 분석 기능 가이드

## 📋 주요 기능 소개

### 1. 향상된 프로필 시스템

#### 기본 정보 입력
- **내 이름**: 개인화된 경험을 위한 필수 입력
- **친구 이름**: 타로 결과에 친구 이름이 포함되어 더욱 의미있는 해석 제공
- **생년월일**: 사주 분석의 기반이 되는 필수 정보
- **성별**: 보다 정확한 해석을 위한 정보

#### 혜택 안내
프로필을 완성하면 다음 기능들이 자동으로 활성화됩니다:
- ✨ 사주 기반 성격 분석
- 🧠 MBTI 스타일 성격 유형 분석
- 💫 친구 이름이 포함된 개인화 타로 해석

---

### 2. 성격 분석 테스트 (10가지 질문)

프로필 입력 후 자동으로 진행되는 성격 테스트:

**질문 카테고리:**
1. 외향성 vs 내향성 (E/I)
   - 대화 주도 성향
   - 새로운 친구 사귀기
   - 모임에서의 태도

2. 계획성 vs 즉흥성 (J/P)
   - 약속 잡기 방식
   - 갈등 해결 접근법
   - 시간 관리

3. 논리 vs 감정 (T/F)
   - 고민 상담 스타일
   - 의견 차이 대응
   - 부탁 수락 방식

4. 활동성
   - 함께 보내는 시간 스타일

**결과:**
- 8가지 성격 유형 중 하나로 분류
- MBTI 스타일 코드 제공 (예: ENTJ, INFP 등)
- 각 유형별 상세 설명

---

### 3. 사주 분석 기능

생년월일을 기반으로 자동 계산:

#### 제공 정보
- **띠**: 12지지 기반 띠 계산
- **사주 정보**: 년주, 월주, 일주
- **오행 분석**: 목(木), 화(火), 토(土), 금(金), 수(水) 중 주 오행
- **성격 특성**: 오행 기반 성격 해석
- **우정 스타일**: 오행별 친구 관계 특성

#### 오행별 특성

**목(木) - 성장형**
- 성격: 성장과 발전을 추구하는 창의적인 성격
- 우정: 친구들의 성장을 도우며 새로운 가능성을 열어주는 관계

**화(火) - 열정형**
- 성격: 열정적이고 활발한 에너지를 가진 성격
- 우정: 따뜻하고 활기찬 분위기를 만드는 관계

**토(土) - 안정형**
- 성격: 안정적이고 신뢰할 수 있는 중심적인 성격
- 우정: 든든한 지지자가 되어주는 깊은 관계

**금(金) - 정의형**
- 성격: 명확하고 결단력 있는 정의로운 성격
- 우정: 솔직하고 진실된 조언을 나누는 관계

**수(水) - 지혜형**
- 성격: 지혜롭고 유연한 적응력을 가진 성격
- 우정: 서로의 감정을 이해하는 공감적인 관계

### 4. 성격 유형 분석 (MBTI 스타일)

10가지 질문 응답을 바탕으로 8가지 유형 중 분류:

#### 유형 목록

**1. 리더형 친구 (ENTJ)**
- 특징: 목표 지향적이고 계획적인 리더십
- 강점: 계획성, 리더십, 문제해결력, 결단력
- 잘 맞는 친구: 조력자형, 분석가형, 모험가형

**2. 조력자형 친구 (ESFJ)**
- 특징: 따뜻하고 배려심 깊은 사교성
- 강점: 공감능력, 배려심, 조화추구, 책임감
- 잘 맞는 친구: 리더형, 중재자형, 예술가형

**3. 모험가형 친구 (ENTP)**
- 특징: 새로운 경험과 아이디어를 즐김
- 강점: 창의성, 유연성, 호기심, 적응력
- 잘 맞는 친구: 리더형, 분석가형, 예술가형

**4. 중재자형 친구 (ENFP)**
- 특징: 열정적이고 긍정적인 에너지
- 강점: 긍정성, 열정, 공감능력, 창의성
- 잘 맞는 친구: 조력자형, 분석가형, 예술가형

**5. 분석가형 친구 (INTJ)**
- 특징: 깊이 있는 대화와 지적 교류
- 강점: 분석력, 독립성, 전략적사고, 통찰력
- 잘 맞는 친구: 리더형, 모험가형, 관찰자형

**6. 수호자형 친구 (ISFJ)**
- 특징: 헌신적이고 신뢰할 수 있음
- 강점: 충성심, 세심함, 인내심, 신뢰성
- 잘 맞는 친구: 조력자형, 관찰자형, 예술가형

**7. 관찰자형 친구 (INTP)**
- 특징: 논리적이고 탐구적
- 강점: 논리성, 객관성, 지적호기심, 독창성
- 잘 맞는 친구: 분석가형, 모험가형, 예술가형

**8. 예술가형 친구 (INFP)**
- 특징: 감성적이고 이상적인 가치 중시
- 강점: 감수성, 창의성, 진정성, 이해심
- 잘 맞는 친구: 중재자형, 조력자형, 수호자형

### 5. 우정 통찰 기능

사주 분석과 성격 유형을 종합하여 제공:

#### 제공 내용
- **종합 인사이트**: 친구 이름을 포함한 개인화된 관계 통찰
- **사주로 본 우정**: 오행 기반 우정 스타일
- **성격으로 본 우정**: MBTI 기반 관계 특성
- **우정 조언**: 3가지 실용적인 조언
  - 서로의 차이 인정과 존중
  - 정기적인 소통의 중요성
  - 함께하는 경험과 추억 만들기

### 6. 타로 결과 개인화

프로필 정보를 활용한 맞춤형 해석:

**친구 이름 입력 시:**
- "OOO님과 XXX님의 우정에 대한 타로 해석입니다"

**내 이름만 입력 시:**
- "OOO님의 친구 관계에 대한 타로 해석입니다"

**친구 이름만 입력 시:**
- "XXX님과의 관계에 대한 타로 해석입니다"

### 7. 접근 경로

#### 홈 화면에서
- 상단 네비게이션 바의 프로필 버튼으로 프로필 입력/수정
- 프로필 완성 후 나타나는 "나의 사주 & 성격 분석 보기" 버튼으로 바로 접근

#### 프로필 모달
- **1단계**: 기본 정보 입력
  - 이름, 친구 이름, 생년월일, 성별
- **2단계**: 성격 테스트
  - 10가지 질문에 응답
  - 언제든 이전으로 돌아가서 수정 가능

## 💡 사용 팁

1. **처음 사용자**: 프로필 입력 시 친구 이름까지 입력하면 더욱 의미있는 결과를 받을 수 있습니다
2. **성격 테스트**: 직관적으로 답변하세요. 정답은 없습니다!
3. **분석 결과**: 사주와 MBTI 분석은 참고용이며, 자신을 이해하는 도구로 활용하세요
4. **친구 관계**: 분석 결과를 친구와 공유하며 서로를 더 잘 이해할 수 있습니다

## 🎨 디자인 특징

- **중앙 모달**: 프로필 입력 시 화면 중앙에 배치되어 집중도 향상
- **단계별 진행**: 기본 정보 → 성격 테스트로 부담 없이 진행
- **시각적 피드백**: 진행률 표시, 호버 효과, 선택 상태 표시
- **반응형 디자인**: 모든 화면 크기에서 최적화된 경험
- **타로 감성 유지**: 퍼플+골드 컬러로 신비롭고 고급스러운 느낌
